Read three distinct integers int a; int b; int c; print smallest among three
----------------------------------------------------------------------------
Input Format:
int a; int b; int c;
----------------------------------------------------------------------------
Constraints:
1<= a <= 109
1<= b <= 109
1<= c <= 109
----------------------------------------------------------------------------
Output Format:
print smallest integer
----------------------------------------------------------------------------
Sample Input 0:
10
15
12
----------------------------------------------------------------------------
Sample Output 0:
10
----------------------------------------------------------------------------
Sample Input 1:
2
4
6
----------------------------------------------------------------------------
Sample Output 1:
2
