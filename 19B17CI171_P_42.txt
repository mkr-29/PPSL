The natural logarithm can be approximated by the following series.
---------------------------------------------------------------------------------
(x-1)/x+(1/2)*[(x-1)/x]^2+(1/2)*[(x-1)/x]^3+(1/2)*[(x-1)/x]^4+.....
---------------------------------------------------------------------------------
If x is input through the keyboard and passed to the function,
write a program to calculate the sum of first seven terms of this series.
---------------------------------------------------------------------------------
Input Format:
An integer number
---------------------------------------------------------------------------------
Constraints:
The entered number should be in the range of 1 to 50,
otherwise program should print Outside the range
---------------------------------------------------------------------------------
Output Format:
Sum of first 7 digits.
---------------------------------------------------------------------------------
Sample Input 0:
3
---------------------------------------------------------------------------------
Sample Output 0:
Log(3) = 1.27
