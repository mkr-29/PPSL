Write a program which “quantifies” numbers. Read an integer “x” and test it, producing the following output:
x greater than or equal to 1000
print “hugely positive”
x from 999 to 100 (including 100,999)
print “very positive”
x between 100 and 0
print “positive”
x exactly 0
print “zero”
x between 0 and -100
print “negative”
x from -100 to -999 (including -100,-999)
print “very negative”
x less than or equal to -1000
print “hugely negative”
---------------------------------------------------------------------------------------------------------------
Input Format:
An Integer
---------------------------------------------------------------------------------------------------------------
Constraints:
The integer entered by the user should be between -5000 to 5000.
If user enter number outside this range it should print "Outside the range"
---------------------------------------------------------------------------------------------------------------
Output Format:
Desired print value.
---------------------------------------------------------------------------------------------------------------
Sample Input 0:
5
---------------------------------------------------------------------------------------------------------------
Sample Output 0:
positive
