Given three points (x1, y1), (x2, y2) and (x3, y3),
write a program to check if all the three points fall on one straight line.
---------------------------------------------------------------------------
Input Format:
-2,2
2,5
6,8
---------------------------------------------------------------------------
where:
First line represents first point.
Second line represents second point.
Third line represents third point.
---------------------------------------------------------------------------
Constraints:
Nothing
---------------------------------------------------------------------------
Output Format:
All 3 points lie on the same line
or
All 3 points do not lie on the same line
---------------------------------------------------------------------------
Sample Input 0:
-2,2
2,5
6,8
---------------------------------------------------------------------------
Sample Output 0:
All 3 points lie on the same line
